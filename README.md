# fortee API スキーマ情報（非公式）

このリポジトリは技術カンファレンス支援プラットフォームであるfortee（ https://fortee.jp/ ）の非公式スキーマ情報リポジトリです。

TypeScriptでAPIを利用する際に使ってください。

## 免責事項
非公式プロジェクト: 本リポジトリおよび公開されているスキーマ情報は、fortee公式が提供するものではありません。

### 無保証
本スキーマ情報の正確性、完全性、最新性について、管理者は一切の保証をいたしません。本情報を使用したことによるトラブルや損害（システム停止、データ破損、法的問題など）について、管理者は一切の責任を負いません。

### 仕様変更
fortee側のアップデートにより、予告なくAPIの仕様が変更され、本スキーマと乖離する可能性があります。

### 利用規約の遵守
APIを利用する際は、fortee本体の利用規約やマナーを遵守し、サーバーに過度な負荷をかけないよう注意してください。

## 注意事項

- optionalはだいぶ間違っているかもです。
- news（お知らせ）は自分の関わっているカンファレンスで使っていないので情報未取得です。

## 使い方

`dist-types/*.ts`をコピペすることが一番単純です。

## 貢献について
forteeのAPIはイベントごとに拡張されたり、新しいエンドポイントが追加されたりすることがあります。 「この型が足りない」「仕様が変わっている」などの修正を見つけた場合は、ぜひPull RequestやIssueでの報告をお待ちしております。
